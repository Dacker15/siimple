Welcome to siimple@css
Introduction to siimple@css
siimple@css is an open source CSS toolkit for building flat and clean web applications and websites. The toolkit has been built with simplicity in mind, and provides a perfect starting point for new web designs.
Let's get started!
Start with siimple@css adding it to your project using npm or yarn:
## Install using NPM
$ npm install --save @siimple/css
## Or install using YARN
$ yarn add --save @siimple/css
Other installation methods
Are you looking for alternative installation methods? Read installation docs.
