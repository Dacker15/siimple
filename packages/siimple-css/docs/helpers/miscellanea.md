#### Rounded corners
> Added in **v3.3.0**.

Add `siimple--rounded` to round the corners of an element.

:::snippet
<div class="siimple--rounded siimple--px-3 siimple--py-3 siimple--bg-light" style="width:200px;">
    Rounded element
</div>
::::
