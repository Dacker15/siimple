What is siimple colors
siimple colors is a flat color palette designed to extend the base colors of the siimple css toolkit. This palette includes fifteen base colors and four variants of each base color (two dark and two light).
You can use this color palette to style your application or webpage, or can be used to develop your custom brand palette.
Getting started
The siimple colors package contains the following files and folders:
Path
Description
/dist
Compiled stylesheets ready to be used in any HTML page
/scss
SCSS files with all color variables and functions
index.scss
Entry SCSS file that builds all CSS class-names
Install using a package manager
You can easily download and install in your project the latest version of siimple colors using npm or yarn: 
$ npm install --save @siimple/colors
$ yarn add @siimple/colors
Now you should reference the dist/siimple-colors.min.css if you are going to use the CSS classes, or the files in the scss folder if you are going to import it in your SCSS project.
Using a CDN
siimple colors can be included in your HTML page from a CDN. Our recomendation is to use jsdelivr to access to the files of this toolkit.
In order to get the latest version, just add the following line inside the <head> tag of each HTML document of your project or website: 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@siimple/colors@{{ site.version }}/dist/siimple-colors.min.css">
